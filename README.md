# AppMataMosquito
Um simples projeto de JavaScript, se trata de um jogo point and click, onde você tem que clickar nas moscar antes que o tempo acabe.

<h4>Instruções</h4>
Ao iniciar será necessario escolher uma dificuldade e clicar para iniciar, cada dificuldade aumenta o tempo que o jogador terá para matar as moscas <br>
Iniciando o jogo você terá de clickar sob a mosca para mata-la até o tempo acabar, caso não mate alguma perdera um coração de vida, e se não restar nenhum você perde o jogo.
